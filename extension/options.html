Supabase Configuration

These settings must match your FocusTube app's Supabase configuration. You can find these values in your .env.local file or Supabase dashboard.

Your Supabase project URL (NEXT_PUBLIC_SUPABASE_URL)

Your Supabase anonymous key (NEXT_PUBLIC_SUPABASE_ANON_KEY)

How to Find Your Supabase Credentials

  1. Open your FocusTube project's .env.local file
  2. Copy the value of NEXT_PUBLIC_SUPABASE_URL
  3. Copy the value of NEXT_PUBLIC_SUPABASE_ANON_KEY
  4. Paste them in the fields above

Note: These are the same credentials your web app uses. The anon key is safe to use in the extension as it only allows authenticated operations.
